[Fit File Analyzer](https://fit-file-analyzer.vercel.app/) is a light-weight webapp that uses [fit-file-parser](https://www.npmjs.com/package/fit-file-parser) to turn .fit files into beautiful charts. This was an experiment and personal challenge I set for myself one weekend - find a way to parse .fit files into usable JSON format.
